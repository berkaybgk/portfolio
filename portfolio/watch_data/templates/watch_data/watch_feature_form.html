About the Watch Price Predictor

How it Works

This is my first end-to-end machine learning project. I am personally interested in the world of watches, and I wanted to see if I could create a tool that would allow me to predict the price of a watch.

About the Data

I first started with scraping data from the Chrono24 website. I ended up with nearly 500k watches for sale. Then, I had to do data cleaning and imputation quite a bit. I ended up with a dataset consisting of nearly 350k watches. I then used several machine learning algorithms to train the model.

Model Results
I have reached a Mean Absolute Error (MAE) of nearly 2000$ and an R^2 of 0.90 using the Gradient Boosting Regressor. The model is then deployed on a web application using Django. The application is hosted on a free-tier EC2 instance.
Data Sources

Chrono24 Website

Finally, it should be noted that there are many more unrepresented features for the watches which causes a big variance for the same models and they require a more detailed scraping process. So the model is open to improvement.
